# esp32s3-wifi_WeatherClock

## 开发环境：
- esp-idf v5.4.1  
- esp_lvgl_port v2.6.0 (lvgl 9.2.2)
- esp32-s3 N16R8  
- 使用PSRAM,外部flash  

## 功能说明：
### v1.1
	- 上电连接wifi，完成SNTP时间同步后显示信息
	- 在LCD上显示时间、日期、温度（目前为英文显示）
	- 每秒更新一次时间，每分钟更新一次天气信息
